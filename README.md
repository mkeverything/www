landing page for [mkeverything](https://mkeverything.com) organization styled as a bash terminal emulator

- available bash commands: `ls`, `cd`, `pwd`, `echo`, `cat`, `whoami`
- available custom commands:
  - `help` — shows custom commands
  - `ru` — translates to russian
  - `en` — translates to english
  - `mail` — allows user to send email to mkeverything for contact
